# Go-to-Market

> Launch strategy and growth levers.

## Launch Phases

### Phase 1: Free Map (Awareness)

- Public map with H3 hexagonal scores visible to all
- Registration gate on DeSO-level detail
- SEO targeting "bästa området att bo i [stad]" queries
- Social sharing of area comparisons

### Phase 2: Freemium Conversion

- Free account shows 5-band percentile and trend direction
- Single-area unlocks (79 SEK per DeSO, 199 SEK per kommun)
- Unlock prompt in sidebar with preview of hidden data

### Phase 3: Subscription

- Monthly (349 SEK) and annual (2,990 SEK) plans
- Full access to exact values, trends, and methodology details
- Positioned for homebuyers in active search (3-6 month lifecycle)

### Phase 4: Enterprise

- Custom tenant with configurable indicator weights
- API access for integration with property platforms
- White-label options for brokerages and banks

## Growth Levers

### Content & SEO
- Methodology transparency drives trust and search ranking
- Public documentation site (this site) serves as content marketing
- Area comparison URLs are shareable and indexable

### Data Network Effect
- More POI data from user contributions improves scoring
- Enterprise customers provide feedback that refines weights
- Annual data refresh creates recurring engagement

### Distribution Partnerships
- Mortgage brokers (pre-approval area research)
- Relocation agencies (corporate moves to Sweden)
- Municipal planning departments (bulk institutional access)

## Key Metrics

| Metric | Description |
|---|---|
| Free → Registered | Conversion at registration gate |
| Registered → Unlocked | First payment (area unlock) |
| Unlocked → Subscriber | Upgrade to full access |
| Monthly → Annual | Plan upgrade |
| Churn | Subscriber cancellation rate |
